# Auri Governance & Compliance Framework
Version: 1.0

---

## Data Sensitivity Categories

### Level 1 – Non-Sensitive
- Logs
- Structural metadata

### Level 2 – Sensitive
- Portfolio holdings
- Financial documents

### Level 3 – Restricted
- API credentials
- Personally identifiable information

---

## Required Controls (Future Commercial)

- Encrypted storage at rest
- Encrypted transport
- Role-based access control
- Immutable audit logs
- Data retention policy
- Export and deletion capability
